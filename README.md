# issen-rs

Reversi solver on Rust

## License

GPLv3

## prerequirements

AVX2 and BMI2 enabled x86\_64 CPU (Intel: Haswell or later, AMD: Excavator or later)

## Setup

prepare evaluation table

```Shell
$ wget https://poyo.me/reversi/table210925.tar.gz
$ tar xf table210925.tar.gz
```

## Run

compile and run

```Shell
$ cargo run --release
```
